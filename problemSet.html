Pseudocode Problems & Solutions

1. Determine if a Number is Positive, Negative, or Zero

Problem Statement: Given an integer num, classify it as positive, negative, or zero.

# Pseudocode: Check sign of a number
if num > 0:
    print("Positive")
elif num < 0:
    print("Negative")
else:
    print("Zero")

2. Calculate the Average of a List of Numbers

Problem Statement: Compute the average of a given list of numbers.

# Pseudocode: Calculate average of a list
tempList = [23, 19, 30, 25, 21]   
total = 0
count = 0

for value in tempList:
    total = total + value
    count = count + 1

average = total / count
print("Average is", average)

3. Find the Maximum in a List of Numbers

Problem Statement: Given a list of integers, find the maximum value among them.

# Pseudocode: Find maximum in a list
numList = [3, 17, 9, 25, 1]  
maxValue = numList[0]        

for num in numList:
    if num > maxValue:
        maxValue = num

print("Maximum value:", maxValue)

4. Count How Many Times a Character Appears in a String

Problem Statement: Count the occurrences of a character in a string.

# Pseudocode: Count character occurrences
text = "banana"
targetChar = "a"
count = 0

for ch in text:
    if ch == targetChar:
        count = count + 1

print("Character", targetChar, "appears", count, "times")

6. Sum All Even Numbers in a Range

Problem Statement: Sum all even numbers in a given range.

# Pseudocode: Sum even numbers in a range
start = 1
end = 10
sumEven = 0

for num in range(start, end + 1):
    if (num % 2) == 0:
        sumEven = sumEven + num

print("Sum of even numbers:", sumEven)

8. Check If a Number is Prime

Problem Statement: Determine if an integer n is prime.

# Pseudocode: Check if number is prime
n = 29
isPrime = True

if n <= 1:
    isPrime = False
else:
    for i in range(2, n):
        if (n % i) == 0:
            isPrime = False
            break

if isPrime:
    print(n, "is prime")
else:
    print(n, "is not prime")

9. Calculate Factorial of a Number

Problem Statement: Compute the factorial of a number.

# Pseudocode: Factorial of n
n = 5
result = 1

for i in range(1, n + 1):
    result = result * i

print("Factorial of", n, "is", result)

10. Convert Celsius to Fahrenheit

Problem Statement: Convert a list of Celsius temperatures to Fahrenheit.

# Pseudocode: Convert Celsius to Fahrenheit
celsiusList = [0, 10, 25, 100]
fahrenheitList = []

for c in celsiusList:
    f = (c * 1.8) + 32
    fahrenheitList.append(f)

print(fahrenheitList)
